# Portfolio

**Dr. K. Kalaivani - Academic Portfolio Website**

## About

This repository contains the academic portfolio website for Dr. K. Kalaivani.

## Getting Started

### Clone the Repository

```bash
git clone https://github.com/kkalaivani-glitch/portfolio.git
cd portfolio
```

### Development

After cloning the repository:

1. Add your portfolio content (HTML, CSS, images)
2. Open `index.html` in your browser to preview
3. Customize the content to match your academic profile
